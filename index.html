Como usar

Via CDN, basta incluir ao final do html, antes do fechamento da tag body, o script:

<script type="module" src="https://cdn.jsdelivr.net/gh/sabia-publisher/paginar/dist/index.es.js"></script> No corpo do html, no local onde deseja que seja renderizado o leitor, utilizar o Web Component conforme abaixo, dentro de um div com propriedade slot="content": <paginate-content id="pagination-el" book-title="Título do livro">
    <div slot="content">
        <p>Conteúdo HTML para paginar</p>
    </div>
</paginate-content>
